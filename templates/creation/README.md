# Compiling in Debug mode
## How to compile
```
cmake . --preset=Debug
cmake --build --preset=Debug
```
## How to execute
```
build/Debug/{{name}}
```

# Compiling in RelWithDebInfo mode
## How to compile
```
cmake . --preset=RelWithDebInfo
cmake --build --preset=RelWithDebInfo
```
## How to execute
```
build/RelWithDebInfo/{{name}}
```
